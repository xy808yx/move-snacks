# Move Snacks

Quick **movement snacks** for kids (~5–10) — a single-file web app that breaks up
long stretches of sitting or screen time with short, randomized bursts of exercise.

Tap once and a slot machine deals you **3, 5, or 7** kid-friendly moves — animal
crawls, jumps, balance, stairs, dance, core — then a read-aloud coach and a big
countdown timer walk you through each one. No equipment, no accounts, works offline.

## Highlights
- **All custom pixel art.** Every exercise is a little posed pixel athlete that
  animates through the movement, drawn by a tiny parametric figure engine — no image files.
- **Built for non-readers.** An optional voice coach reads each move aloud.
- **Apple-Fitness-style** activity ring, day streaks, and a pixel "buddy" you collect each session.
- **One file.** All art, audio, and logic live in `index.html`.

## Run it
Open `index.html` in a browser — that's the whole app. Or play the live version:

**https://xy808yx.github.io/move-snacks/**

Add it to your home screen (it's an installable PWA) for a full-screen, app-like experience.
